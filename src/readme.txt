Prerequisite:
    MATLAB need to be installed.
    libsvm need to be installed.

Pre-Processing:
    pre_processing.m : Run this in MATLAB to dump train, test files in format compatible with libsvm library.
    
Evaluation:
    Use easy.py to get the evaluations.
    Usage: easy.py <train_file> <test_file>
       Ex: ./easy.py ~/workspace/free_lancer/projects/kruthika/data/train_test/train.txt ~/workspace/free_lancer/projects/kruthika/data/train_test/test.txt
